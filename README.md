## 📖 게임 소개

'나비효과(The Butterfly Effect)'는 환경 문제를 주제로 제작한 인터랙티브 웹 게임입니다.

플레이어는 집안에 있는 다양한 자원 낭비 요소를 순서에 따라 처리하며 환경 보호의 필요성과 실천 방안을 배우게 됩니다. 단계별 엔딩 장면은 작은 행동이 큰 변화를 가져온다는 메시지를 전달하며, 플레이어가 다음으로 처리할 요소를 안내하는 역할을 합니다.

## 🕹️ 게임 방법

1. 방 안의 오브젝트를 클릭하여 상호작용합니다.
2. 정해진 순서대로 자원 낭비 요소를 처리하며, 필요 시 미니게임을 클리어한다.
3. 중간에 현관문을 통해 나가면 현재 진행 단계에 따른 엔딩을 확인할 수 있습니다.
4. 모든 자원 낭비 요소를 처리하면 최종 엔딩을 볼 수 있습니다.
